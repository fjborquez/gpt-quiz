## Installation

```bash
brew tap rsaryev/gpt-quiz https://github.com/rsaryev/gpt-quiz.git
brew install gpt-quiz
```

## Usage

Start quiz with a prompt
```bash
gpt-quiz start "golang interview questions"
```
Reset configuration
```bash
gpt-quiz reset
```
